About Me

I’m Aaron Sanga, a 2nd year student at Ontario Tech University. I am currently completing my 4 year program in Technology Management BIT with an increasing interest in combining my interest for both business and technology into a career that I can be proud of in the future.

Here are my recent projects

Project 1

Random Cipher Generator

This project was completed in Intro to Object Oriented Programming in which the program I have created defines and executes various ciphers including a ceasar cipher, transposition cipher, product cipher, substitution cipher, playfair cipher, and an RSA cipher. The program takes an input from the user and randomly selects one of these ciphers to use on the user generated text.

Project 2

BMI Calculator

This is another project created in Into to Object Oriented Programming in which a seperate tab is opened and asks for the users weight in kilograms and height in meters in order to calculate their Body Mass Index. The program will then determine based on real life statistics whether the inputs provided by the user and their resulting BMI classifies them as underweight, overweight, or healthy.
